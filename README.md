# 📸 Responsive Image Gallery

## 📖 Overview

This project is a Responsive Image Gallery created using **HTML**, **CSS**, and **JavaScript**. It displays images in a responsive grid layout and provides an interactive lightbox with Previous and Next navigation. Users can also filter images by category, making the gallery more organized and user-friendly.

---

## 🚀 Features

* Responsive image gallery layout
* Lightbox image preview
* Next and Previous image navigation
* Hover zoom effect
* Smooth CSS transitions
* Category filters (All, Nature, City, Animals)
* Mobile-friendly responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension (recommended).
4. Right-click **index.html** and select **Open with Live Server**.
5. The image gallery will open in your browser.

---

## 📱 Project Features

* Displays images in a responsive grid.
* Opens images in a lightbox when clicked.
* Navigate images using **Next** and **Previous** buttons.
* Zoom effect on image hover.
* Smooth transitions and animations.
* Filter images by category.
* Works on desktop, tablet, and mobile devices.
